# 主要区别
## 1. 功能定位不同
t00lsQD.py：长期自动化脚本，包含智能调度系统，可以365天自动运行
t00ls签到.py：交互式单次脚本，需要用户手动选择操作

## 2. 执行方式不同
**特性	t00lsQD.py	t00ls签到.py**
执行模式	自动循环执行	手动选择功能
时间调度	有（固定11:30-12:00）	无
等待机制	智能等待到目标时间	立即执行

## 3. 核心功能对比
**t00lsQD.py的主要特色：**
1、时间调度系统
calculate_next_execution_time()  # 计算下次执行时间
wait_until(target_time)          # 智能等待
2、循环执行机制
main() 中的 while 循环，支持365天运行
每次执行后自动安排下一次（明天同一时间段）
3、完整的执行日志
记录每次执行的详细信息、响应数据
